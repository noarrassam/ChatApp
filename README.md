# iOS MidTerm Project


## Description
It’s a chat app for social networking as we’re living in pandemic situation to people can interact with each others.

## Functionalities

* Firebase
* Firestore
* Adobe XD and Zeplin
* Cocoapods 
* Login and register
* Email Verification
* Forgot Password
* Authentication
* Send messages
* Creating session & Logout
* User Profile
* Setting
* Logout
