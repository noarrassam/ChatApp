# ChatTeam Project


## Description
It’s a chat app for social networking as we’re living in pandemic situation to people can interact with each others.

## Functionalities

* Firebase
* Firestore
* Adobe XD and Zeplin
* Cocoapods 
* Login and register
* Email Verification
* Forgot Password
* Authentication
* Send messages
* Creating session & Logout
* User Profile
* Setting
* Logout

## Citation
* Udemy Course, [iOS 13 & Swift 5: Chat App With Firestore | MVVM](https://www.udemy.com/course/ios-13-swift-5-chat-app-with-firestore-mvvm/).
